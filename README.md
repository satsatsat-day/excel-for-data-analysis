# excel-for-data-analysis
이 레포지토리는 데이터 분석 직무 준비를 위해
**엑셀 실무 연습**을 기록한 학습 저장소입니다.

## 학습 내용
※ 아래 파일들은 강의 자료를 바탕으로 직접 실습하며 내용을 수정·추가한 개인 학습용 파일입니다.

## 실습 파일: 아이스크림 매출 현황.xlsx
### 자동채우기 / 빠른채우기
- 자동채우기: 패턴이 있는 데이터 입력을 빠르게 채우기 [+ 더블클릭] or [+ 드래그]
<br/>(날짜를 자동채우기할 때, 셀 우측하단 모서리에 옵션이 있음. 평일, 주말 등으로 날짜 지정 가능)
- 빠른채우기: 기존 데이터를 기준으로 문자열 분리/결합 자동 처리 [Ctrl + E]
- 실습 파일: 아이스크림 매출 현황.xlsx

## 실습 파일: 과일_재고 현황.xlsx

### 셀에 폰트 크기 맞춤
- 셀 서식(Ctrl + 1) → [맞춤] → [텍스트 조정] → '셀에 맞춤' 체크
- 폰트 크기가 너무 작아지면, 열 너비 넓히기

### 숨기기 / 그룹
- 숨기기: 숨겨진 열 우클릭 → '숨기기 취소'
- 그룹 기능: 숨기고 싶은 범위(행 또는 열) 그룹화해서 숨기기
  - 그룹화: ALT + SHIFT + 오른쪽 방향키
  - 그룹 해제: ALT + SHIFT + 왼쪽 방향키

### 테두리
- 테두리 하고싶은 범위 드래그 한 후, ALT → H → B → A 순으로 누르기
- 테두리 제거: Ctrl + SHIFT + '-'=

### 그림 추가하기
- 인터넷에 있는 사진을 저장하지 않고 엑셀에 삽입 가능 (인터넷 이미지를 엑셀로 바로 드래그)
- ALT 누르면서 사진 크기 조정하면 쉽게 맞춰서 삽입 가능
